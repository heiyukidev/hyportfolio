My Setup

This is my Work environment and all the technologies I use to develop web applications.

Since I do mostly javascipt developpement most of these tools are geared towards that.

Software

Ubuntu: Ubuntu is an open source software platform. It's a fancy way to say Operating system and I use it daily. Most of server are linux powered machines, So I feel at home when I ssh into a server.
Atom: Atom is hackable tex editor. I configure snippets all the time for more productivity. I use Atom for everything! It has good support for a lot of languages but I mostly use it for Javascript.
Trello: Trello is a way to organize my work. It's very simple and flexible. Most people don't realize how important it is to visualize how much work you have done and still have to do. It helps you better manage your time.
NPM: Arguable the best this ever. NPM stands for Node Package Manager. It's a registry of modules and packages. I use NPM to install everything and I contribute by Publishing my own modules too.
Bower: Bower is a dependency Manager for The Web. I use it for front end Applications to install my libraries. It's also available on NPM so you can use both on a single project.
Yeoman: Yeoman is a tool to generate projects. I use it to quickly setup Javascript environments because a lot of them take time to properly setup.
